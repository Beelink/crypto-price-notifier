# MexcPrice | crypto-price-notifier

Built on top of the `mexc-api-sdk` and `node-telegram-bot-api`

### Start:

`ts-node` is needed

```bash
yarn
```

```bash
yarn start
```

### Possible improvements:

1. Add option to send message on price check
